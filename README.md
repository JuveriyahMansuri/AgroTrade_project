The Main Aim of E-AGROTRADE Application is to provide a Generalized platform to Wholesalers & Retailers where they can buy/sell Fruits/Vegetables by conducting E-Auction.




## ðŸ›  Tech Stack

**Client:** Flutter

**Server:** Django , SQLite


## ðŸ’» Features

- 3-step Onboarding Process with OTP

- Add Products
- Place Bids

- Conduct E-Auction on basis of start time and end time

- System would provide the winner



- See Live Auction Status

- Manage Payment Status

-  Manage & Check Order/Delivery Status

- Manage Customers(Retailers)
